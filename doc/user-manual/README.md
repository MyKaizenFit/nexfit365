# 🎮 Manual de Usuario - Nex-Fit

## 👋 Bienvenido a Nex-Fit

Nex-Fit es tu compañero personal para alcanzar tus objetivos de fitness y bienestar. Esta aplicación te permite gestionar entrenamientos, nutrición, progreso y logros de manera integral y personalizada.

## 🎯 ¿Qué Puedes Hacer?

### 🏋️ **Gestión de Entrenamientos**
- Crear y personalizar programas de entrenamiento
- Seguir rutinas predefinidas o crear las tuyas propias
- Registrar sesiones de entrenamiento con detalles
- Hacer seguimiento de tu progreso físico

### 🥗 **Planificación Nutricional**
- Crear planes de nutrición personalizados
- Calcular macronutrientes y calorías
- Seguir tu ingesta diaria de alimentos
- Recibir recomendaciones nutricionales

### 📊 **Seguimiento del Progreso**
- Registrar fotos de progreso
- Hacer seguimiento de medidas corporales
- Visualizar gráficos de evolución
- Establecer y alcanzar metas

### 🏆 **Sistema de Logros**
- Desbloquear badges por objetivos cumplidos
- Seguir tu historial de logros
- Competir con otros usuarios
- Mantener motivación constante

### 🔔 **Notificaciones Inteligentes**
- Recordatorios de entrenamientos
- Alertas de nutrición
- Notificaciones de logros
- Recordatorios personalizados

## 🚀 Primeros Pasos

### 1. **Crear Cuenta**
- Ve a la página de registro
- Completa tu información personal
- Establece tus objetivos de fitness
- Configura tu perfil

### 2. **Configurar Perfil**
- Sube una foto de perfil
- Establece tu nivel de experiencia
- Define tus objetivos principales
- Configura preferencias de notificaciones

### 3. **Crear Primer Entrenamiento**
- Selecciona un programa predefinido
- Personaliza según tus necesidades
- Establece horarios de entrenamiento
- Comienza tu primera sesión

### 4. **Configurar Nutrición**
- Define tu plan nutricional
- Establece objetivos de calorías
- Configura preferencias alimentarias
- Comienza a registrar tu alimentación
